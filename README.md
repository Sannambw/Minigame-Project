# 🕹️ Klickspel

Ett enkelt klickspel där spelaren ska klicka så många gånger som möjligt på 60 sekunder.

##  Hur spelet fungerar
- Klicka på knappen för att få poäng. 
- Tiden kommer börja räknas ner från 60 sekunder när du börja klicka 
- När tiden är slut kommer din slutpoäng visas 
- Skriv ditt namn och spara ditt resultat
- Efter sparat resultat kommer resultattavlan komma upp och de 10 bäst kommer att visas 

## Funktioner
- Poängräkning i realtid 
- Nedräkning (60 sekunder)
- Gameover 
- Spara resultat via API (Zapier + Google Sheets)
- Hämtar och visar resultattavla
- Sorterar högsta poäng först
- Bara visa de 10 bästa
